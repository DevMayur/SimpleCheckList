SimpleCheckList MCP Server - Global Usage Guide
===============================================

Your SimpleCheckList MCP Server is ALREADY GLOBALLY AVAILABLE!
You can use it on any PC worldwide right now.

QUICK START ON ANY PC
=====================

1. Install Docker (if not already installed)
   - Windows: Download Docker Desktop from docker.com
   - Mac: Download Docker Desktop from docker.com  
   - Linux: Install docker via package manager

2. Pull and run your MCP server (ONE COMMAND):
   docker run -p 8355:8355 mayurkakade/mcp-server:latest

3. Access the API:
   http://localhost:8355/api/health

CLAUDE DESKTOP INTEGRATION (ANY PC)
===================================

Add this to Claude Desktop configuration on any computer:

{
  "mcpServers": {
    "simple-checklist": {
      "command": "docker",
      "args": [
        "run", "--rm", "-i", 
        "-p", "8355:8355", 
        "mayurkakade/mcp-server:latest", 
        "mcp"
      ],
      "env": {
        "API_BASE_URL": "http://localhost:8355/api"
      }
    }
  }
}

PRODUCTION DEPLOYMENT (ANY PC)
==============================

For persistent data on any PC:

docker run -d --name simplechecklist \
    -p 8355:8355 \
    -v simplechecklist_data:/app/data \
    --restart unless-stopped \
    mayurkakade/mcp-server:latest

USAGE EXAMPLES ON ANY PC
========================

# Basic health check
curl http://localhost:8355/api/health

# Create a project
curl -X POST http://localhost:8355/api/projects \
  -H "Content-Type: application/json" \
  -d '{"name":"My Project","description":"Testing on new PC"}'

# List projects  
curl http://localhost:8355/api/projects

DIFFERENT STARTUP MODES
=======================

# Backend API only
docker run -p 8355:8355 mayurkakade/mcp-server:latest backend

# MCP server only (for Claude Desktop)
docker run --rm -i mayurkakade/mcp-server:latest mcp

# Both services (default)
docker run -p 8355:8355 mayurkakade/mcp-server:latest both
