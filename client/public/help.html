New! SimpleCheckList now supports Cursor IDE with full MCP integration.

Step-by-Step Setup:

Step 1: Open Cursor Settings
  • Press Cmd/Ctrl + , or go to Cursor â†’ Settings
Step 2: Navigate to MCP Settings
  • Click on "MCP" in the left sidebar
Step 3: Add New MCP Server
  • Click "New MCP Server" button
Step 4: Configure the Server
  • Server Name: simple-checklist
  • Command: node
  • Arguments: /path/to/SimpleCheckList/mcp-server/index.js
  • Environment Variables:
    • Key: API_BASE_URL
    • Value: http://localhost:8355/api
Step 5: Save and Test
  • Click "Save" to add the server
  • Restart Cursor to activate the MCP server
  • Test by asking: "Create a new project called 'Test Project'"

ðŸ”§ Troubleshooting

Common Issues:

"Server not responding"
  • Check if backend is running: curl http://localhost:8355/api/health
  • Verify port 8355 is available
  • Restart your AI tool after adding the MCP server
"Command not found"
  • Use full path to node executable
  • Verify the path to mcp-server/index.js is correct
  • Check file permissions
